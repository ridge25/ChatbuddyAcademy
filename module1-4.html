üìà Module 1.4: Increased Awareness

Now since your chatbot has some personality, we'll start making it aware of things such as time of day (i.e. Morning, Afternoon, Evening), the current date & current time.

Greeting Awareness

Let's make your bot aware of what time it currently is. Note that this relies what time your current operating system (OS) is functioning on...

First, underneat 'import random', we need to add 'from datetime import datetime.' Your very first two lines of code should look like this:

import random
from datetime import datetime

This doesn't have any functionality yet, but is the first step in making sure your bot is aware of the time of day!

Now, we can get into the code structure! We need to add the time criteria that it will follow to determine which greeting to use. Let's use this code:

# Time of day greeting
current_hour = datetime.now().hour
            
if current_hour < 12:
    time_greeting = "Good morning"
elif current_hour < 18:
    time_greeting = "Good afternoon"
else:
    time_greeting = "Good evening"

üî• Pro Tip

Using a '#' negates code and is used for labeling. As your bot starts to become more advanced, it would be smart to start labeling. If you ever need to go back into your code to edit a certain part, the label will help you find it easily!

ü•á Step 1: Combine Greetings

We want to make sure that while your bot recognizes the time of day, it also still pulls greetings randomly from the list we've already made. To do this, we'll write some code that looks like this:

# Combined greeting
print(f"ChatBuddy: {time_greeting}, {user_name}! {random.choice(greetings)} Ask me anything. üòÑ")

Your chatbot will now understand to 1. Figure out what time of day meets the criteria and 2. Pull a random greeting from the list.
